# FPS
UnityでFPSもどきを作成する
FPSというより、ゾンビアクションゲームになるかもしれません。(未定)

<dl>
  <dt>操作方法</dt>
  <dd>上下左右：WASD　または　↑↓←→</dd>
  <dd>攻撃　　：左クリック　長押しで連射</dd>
  <dd>武器変更：右クリック</dd>
  
  <dt>使用可能武器</dt>
  <dd>銃　：20発、チャージ(リロード)タイムがある。</dd>
  <dd>ボム：爆発するまで次のものが投げれない。</dd>
  
  <dt>音</dt>
  <dd>撃った時、投げた時、爆発するときに出るようにした。</dd>
</dl>
